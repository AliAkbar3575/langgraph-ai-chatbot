## Conversational AI Chatbot with Memory Using LangGraph and Streamlit
---
#### Tools used:
- LangChain
- LangGraph
- Groq
- Llama-3.1
- MemorySaver
- Streamlit
---
This project is a conversational AI chatbot developed using LangChain, LangGraph and Streamlit. The system integrates the Groq hosted Llama 3.1 language model to generate intelligent and context-aware responses in real time. The chatbot is designed with a graph-based workflow architecture where conversation states are managed using LangGraph’s StateGraph and MemorySaver. This enables the application to maintain conversational memory and support context-aware multi-turn interactions. User and AI messages are stored as chat states, allowing the model to remember previous conversations and provide more coherent responses. The frontend interface is built with Streamlit, featuring an interactive chat UI, conversation history management, and sidebar controls for resetting chat memory. The application demonstrates the integration of Large Language Models (LLMs), conversational memory, and modern AI workflow orchestration to create a responsive and user-friendly AI assistant.
